# ONE Pass Privacy Notice — Draft

Requires qualified Korean privacy/legal review. ONE Pass separates ordinary travel preferences from sensitive identity. Before any identity use, the user must see the exact fields, recipient/provider, purpose, retention, overseas processing where applicable, and deletion controls. Identity is never sent to analytics or AI. Passport images, NFC contents and biometric templates are not stored in the MVP. Consent is specific, versioned and revocable where applicable.
