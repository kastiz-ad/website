# ONE Interactive Revision Engine V1

The final review page has a compact multilingual command box directly above approval. It supports multiline input, Enter submission, Shift+Enter newline, accessible labeling, loading/error states, and English/Korean/Spanish copy.

The revision engine classifies 27 revision intents, updates only affected mission sections, preserves unaffected data, recalculates dependencies, revalidates completeness, records the configured provider, and returns a localized added/changed/removed/verification/approval summary. It never executes externally.

Pathway suggestions now sit above the revision field. Selecting one converts it into a revision command; material pathway changes revalidate dependencies and clear prior approval while unaffected milestones remain intact.
