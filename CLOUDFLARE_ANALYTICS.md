# Cloudflare Analytics

Cloudflare Web Analytics may measure aggregate page traffic at the hosting layer. The explicit frontend beacon adapter remains disabled until a unique beacon token and consent/legal decision are configured, preventing duplicate loading.

Founder reports are available in Cloudflare Dashboard → Web Analytics. Cloudflare is strongest for aggregate traffic and performance; GA4 provides consented product-event funnels. The two systems have different counting methods and should not be expected to match exactly.
