# ONE Multilingual Conversation Engine V1

The server builds a language profile before mission decomposition. OpenAI receives the requested conversation language, interface locale, tone context, and local service region without an English-first translation stage. Code-switched messages retain place names, brands, professional vocabulary, emotional intent, and the dominant or explicitly requested language. Bilingual output is supported when requested.

Reviewed native styles include friendly, secretary, business, family, and institutional Korean; neutral, conversational, business, and concise Spanish; and conversational, executive-assistant, business, support, and operational English. Other model-supported languages use the existing interface locale and must not be advertised as reviewed UI locales.
