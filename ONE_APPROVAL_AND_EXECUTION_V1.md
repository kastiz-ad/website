# ONE Approval and Execution V1

Levels: 0 read/reason; 1 prepare; 2 reversible draft; 3 external action; 4 financial/high consequence. Levels 3–4 require explicit, current approval bound to exact action terms. Review shows recipient, shared data, total cost, cancellation terms and risks.

The V1 orchestrator models queues, dependency order, receipts, blocking and recovery. It cannot book, reserve, pay, send email, contact providers, deploy or perform destructive repository actions. Even an approved action remains blocked when a production provider is unavailable.
