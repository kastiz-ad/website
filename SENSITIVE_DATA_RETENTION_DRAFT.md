# ONE Pass Sensitive Data Retention — Draft

Requires Korean privacy/legal review. Retain only fields necessary for an approved purpose. Support `always ask`, `device confirmation every time`, and `use once and delete`. Expired/replaced passports become unusable. Deletion revokes vault access and provider connections; legally/security-required privacy-safe records may be retained only for a documented period. No passport image, NFC payload, biometric, CVV, card number, bank password, provider password, or raw access token belongs in ordinary storage.
