# Trusted User Journey V1

The reference path is Homepage → guest/account → progressive profile → mission → necessary follow-ups → comparison → explained recommendation → customization → exact data-release preview → secure confirmation demo → safe simulated execution → receipt → history → Trust Center.

The Tokyo reference demo lives at `trusted-demo.html`. It uses explainable provider scoring, a confidence score, exact shared/excluded fields, ten-minute single-use permission, truthful secure-demo language, a demo provider, and a privacy-safe receipt. It never contacts a provider.

Returning users use `life.html`; guests see device state, and logged-in users can use protected backend data once staging is configured. All main controls are keyboard reachable, focus-visible and responsive at 390px.
