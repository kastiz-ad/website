# ONE Dynamic Roles V1

ONE selects roles from the mission request; users never choose them manually. Supported roles include travel planner, executive assistant, personal secretary, researcher, writer, editor, programmer, technical advisor, project manager, restaurant planner, event coordinator, family coordinator, education coordinator, healthcare logistics assistant, provider researcher and transport coordinator.

Specialists return structured work only. They cannot execute external actions. `ONE_ORCHESTRATOR` retains mission control and `APPROVAL_AND_RISK_SPECIALIST` protects consequential boundaries. Simple missions use few specialists.
